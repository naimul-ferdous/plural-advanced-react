# advanced-react

Advanced React.

## Building and running on localhost

First install dependencies:

```sh
npm install
```

To run:

```sh
npm start
```

To create bundle.js:

```sh
npm run webpack
```

To test:

```sh
npm run test
```

## Running

Open the url `https://localhost:8080` in your browser
